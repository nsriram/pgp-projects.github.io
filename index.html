PGP Data Science & AI — Project Portfolio

End-to-end projects spanning Python, Machine Learning, Deep Learning, NLP, Computer Vision, MLOps, and Generative AI.

01
Intro to Python

FoodHub Order Data Analysis

Exploratory data analysis on a food aggregator platform to understand demand patterns, restaurant performance, and customer behaviour across cuisines and time-of-week.

Outcome: Top 5 restaurants account for 35% of revenue; American cuisine leads preference; weekends drive significantly higher order volumes; 59% of customers are repeat users.
02
Machine Learning

AllLife Bank — Personal Loan Campaign

Identifies retail banking customers most likely to accept a personal loan offer using decision-tree classifiers with class-balancing techniques on an imbalanced dataset.

Outcome: Post-pruned decision tree achieves 99.7% recall on training data; class-weight balancing and pre-pruning compared to find the optimal trade-off between precision and recall.
03
Advanced Machine Learning

EasyVisa — Visa Approval Prediction

Predicts employment-based visa certification outcomes for OFLC using ensemble methods (Random Forest, XGBoost, Bagging) to help shortlist high-probability candidates.

Outcome: Tuned XGBoost selected as best model; job experience, high-school education level, and Asian origin are the strongest predictors of visa certification.
04
Neural Networks

ReneWind — Wind Turbine Failure Prediction

Uses neural networks with regularisation techniques to predict generator failures in wind turbines from sensor data, enabling cost-effective predictive maintenance.

Outcome: Dropout-regularised model achieves 0.92 validation recall with only 19 false negatives; batch normalisation + dropout reduces false positives from ~100 to 20, dramatically cutting unnecessary inspections.
05
NLP with Generative AI

Merck Manuals — Medical Q&A with RAG

Builds and evaluates three medical Q&A approaches — baseline LLM, prompt engineering, and Retrieval-Augmented Generation — on content from the Merck Manuals to ensure accurate, grounded answers.

Outcome: RAG with fine-tuned prompting raises groundedness from 3/5 to 4/5; all test queries achieve high relevance; RAG outperforms the baseline LLM significantly across evaluation metrics.
06
Intro to Computer Vision

SafeGuard — Workplace Helmet Compliance Detection

Develops image-classification models to automatically detect whether workers are wearing helmets, comparing a custom CNN architecture against transfer learning with VGG-16.

Outcome: VGG-16 with a feed-forward head reaches 98.9% validation accuracy vs. 92.6% for the custom CNN — a 6.3 percentage-point gain that translates directly to fewer undetected safety violations.

Model Deployment

SuperKart — Sales Revenue Predictor

Builds and deploys a production-grade product sales forecasting system for a retail chain, integrating trained regression models with a Streamlit front-end in a containerised deployment pipeline.

Outcome: End-to-end deployment achieved — model predicts revenue across stores, store types, and cities; fully functional Streamlit app containerised and ready for production serving.

Intro to SQL

New Wheels — Vehicle Sales SQL Analysis

Uses SQL queries to analyse vehicle sales, order trends, customer satisfaction, and revenue performance for an online vehicle subscription business.

Outcome: SQL-based reporting pipeline delivers quarterly business insights on revenue, top-selling vehicles, and customer satisfaction scores to guide management decision-making.

Applied Statistics

ENews — Landing Page A/B Test

Applies hypothesis testing (t-test, z-test, chi-square, ANOVA) to compare old and new landing pages for a digital news platform, measuring engagement time and subscription conversion.

Outcome: New page shows significantly higher time-on-page and conversion rate (p < 0.05); language preference does not drive conversion — validating the new design for a full rollout.

Visit With Us — Tourism Package Prediction with MLOps

Predicts customer likelihood of purchasing a Wellness Tourism Package while building a complete MLOps pipeline with GitHub Actions CI/CD for automated model training, versioning, and deployment.

Outcome: Full MLOps pipeline operational — data registration, preparation, training, and Hugging Face deployment stages automated end-to-end with continuous model update workflows.

Advanced GenAI for NLP

NewsFindr — Agentic Personalised News Retrieval

Builds a multi-tool AI agent combining SQL queries, web search APIs, and an LLM to deliver real-time personalised news with multi-stage credibility and relevance filtering.

Outcome: All four objectives met — real-time retrieval, credibility validation, engagement enhancement, and streamlined consumption; sequential tool execution prevents race conditions; all test cases validated.
12
Capstone

FoodHub AI — Customer Service Chatbot (Capstone)

Designs and deploys a production AI chatbot integrating an LLM, a SQL database agent, and multi-layer guardrails to handle customer order queries autonomously and escalate edge cases to human agents.

Outcome: Refined prompting improves accuracy by 90% (2.22 → 4.22 / 5) and clarity by 60% (2.78 → 4.44 / 5); SQL agent achieves exact matches across all 10 order columns; 7/7 test queries answered correctly; deployed to Hugging Face with CI/CD.
